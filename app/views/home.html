Welcome to

A collaborative travel itinerary application

Numinous aims to provide our users a simple method to generate upcoming trips and vacations. Our tool helps by organizing and displaying trip information in various views. We allow offer the ability to share and invite other users to any create trips.

Features

AngularJS

AngularJS was chosen because it let us extend HTML vocabulary in our application. This resulted in creating an enironment that was readable, expressive, and quick to develop.

SailsJS

Sails is a popular MVC framework used to develop Node.js applications. Sails helped us write our application entirely in Javascript. Other great features about Sails were: easy database connections, auto-generate REST APIs, and reusable security policies

Google Map's API

Google's web APIs here used to incorporate rich mapping features that could provide accurate directions, as well as, useful information about an establishment. All the maps were customized to meet the basic requirements for a traveler.

OpenWeatherMap API

This API was simple and free to use. It offers our users current weather data for any location on Earth. Further development will incorporate 5day forecasting to help our travelers plan accordingly.
